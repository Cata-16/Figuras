# Figuras

Este proyecto muestra figuras geométricas tridimensionales (cubo, cilindro y esfera) creadas con Three.js, integradas en una página web con un diseño moderno usando Bootstrap.  

Cada figura se representa en un entorno 3D interactivo animado, tanto en la parte superior (banner principal) como dentro de tarjetas informativas con su respectiva descripción.

# Tecnologías utilizadas

- HTML5  
- CSS
- JavaScript
- Three.js
- Bootstrap
